Media processing library (MPL)

Основное назначение:
    Маршрутизация медиа-потоков по различным технологиям

Функционал:
    - Получение медиа-данных от: 
        - v4l2-устройств;
        - rtsp-камер;
        - rtmp-сереров;
        - локальных медиа-файлов (mp3, mkv);
        - http/https-медиа-потоков;
        - hls-потоков;
        - микрофона (alsa, pulsa);
        - vnc-серверов;
    - Передача медиа-данных;
        - на rtmp-сервер;
        - в локальный файл (mp3, mkv);
        - в динамик (alsa, pulsa);
        
    - Транскодирование и конверсия аудио-видео потоков;
    - Управление устройствами камерами:
        - через интерфейс v4l2;
        - по протоколу visca (serial-port);
    - Каскад эхокомпенсации (на базе webrtc-audio-processing);
    - Микширование аудио потоков;
    - Детектор уровня звучания для аудио-потоков;
    - Композирование видео-потоков;
    - Обработка видео-потоков:
        - Наложение текста
        - Наложение примитивной графики (прямоугольники, линии, овалы и т.д.);
        - Наложение CV-фильтров;

Сборка:
    #scripts/install-build-debs.sh
    $mkdir -p build && cd build && cmake -DCMAKE_BUILD_TYPE=Debug .. && make
